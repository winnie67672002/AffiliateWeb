# 部署指南：複製這套部落格到你自己的網站

## 第一步：安裝環境（只需做一次）

需要先安裝：
- [Node.js 20+](https://nodejs.org/)
- [Git](https://git-scm.com/)

---

## 第二步：取得程式碼

```bash
# 把專案複製到本機（請換成實際的 GitHub/來源網址）
git clone https://github.com/你的帳號/my-blog.git

# 進入資料夾
cd my-blog

# 安裝套件
npm install
```

---

## 第三步：改成你自己的名字與網域

以下檔案需要修改，全部用「搜尋取代」把這些值換掉：

| 找這個 | 換成 |
|---|---|
| `goodpickslab.com` | 你的網域（例如 `myblog.com`） |
| `Good Picks Lab` | 你的網站英文名稱 |
| `選好研究所` | 你的網站中文名稱 |
| `AW-17789304314` | 你的 Google Analytics ID（或直接刪掉 GA 相關程式碼） |

### 要改的檔案清單

**`app/layout.tsx`**
```
metadataBase: new URL('https://goodpickslab.com')  → 你的網域
title: 'Good Picks Lab'                             → 你的名稱
siteName: 'Good Picks Lab'                          → 你的名稱
選好研究所（兩處 schema）                            → 你的名稱
AW-17789304314（兩處）                              → 你的 GA ID
```

**`app/sitemap.ts`**
```
const BASE_URL = 'https://goodpickslab.com'  → 你的網域
```

**`components/Header.tsx`**
```
選好研究所 Good Picks Lab  → 你的網站名稱
字母 "G"（logo 圖示）      → 你名稱的第一個字母
```

**`components/Footer.tsx`**
```
Good Picks Lab（兩處）  → 你的名稱
```

**`lib/gtag.ts`**
```
AW-17789304314  → 你的 GA ID（沒有 GA 就把這個檔案的 export 留空即可）
```

---

## 第四步：在本機預覽

```bash
npm run dev
```

打開瀏覽器前往 `http://localhost:3000` 確認樣式與名稱都正確。

---

## 第五步：部署到 Vercel（免費）

```bash
# 安裝 Vercel CLI
npm install -g vercel

# 登入並部署
vercel

# 之後每次更新只要執行
vercel --prod
```

部署後在 Vercel 後台 → **Domains** 頁面綁定你自己的網域即可。

---

## 第六步（選用）：換成你自己的文章

文章放在 `content/posts/` 資料夾，格式為 Markdown（`.md`），
每篇文章開頭需要有：

```markdown
---
title: 文章標題
date: 2026-01-01
description: 文章描述
category: 3C產品
---

文章內容寫在這裡...
```

把原有的 `.md` 檔刪掉，換成你自己的文章就完成了。

---

## 完整改動一覽（快速版）

如果你熟悉命令列，可以用這個指令一次替換所有網域（macOS/Linux）：

```bash
# 替換網域（把 myblog.com 換成你的）
find . -not -path '*/node_modules/*' -not -path '*/.git/*' \
  -type f \( -name "*.ts" -o -name "*.tsx" \) \
  -exec sed -i '' 's/goodpickslab.com/myblog.com/g' {} +

# 替換網站名稱
find . -not -path '*/node_modules/*' -not -path '*/.git/*' \
  -type f \( -name "*.ts" -o -name "*.tsx" \) \
  -exec sed -i '' 's/Good Picks Lab/你的名稱/g' {} +
```

Windows 用 VS Code 的「全域搜尋取代」(Ctrl+Shift+H) 做同樣的事即可。
